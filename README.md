# 🧠 InvestEtec-Vitrine de Talentos e Investimentos para estudantes da ETEC

## Uma Iniciativa em ação da inovação no mercado

Bem-vindo ao repositório do projeto Invest Etec, uma aplicação web educacional desenvolvida para ser uma vitrine de exposição de projetos acadêmicos de alunos da ETEC Sylvio de Mattos Carvalho e consolidando potenciais investidores, conectar o talento técnico estudantil com oportunidades reais de mercado e investimento.

---

## 💡 Objetivo 

Desenvolver uma plataforma digital para a exposição de projetos acadêmicos, conectando alunos da ETEC a investidores e empresas para gerar oportunidades no mercado de trabalho.

- **🚀 Fomentar**:a visibilidade dos projetos técnicos desenvolvidos na instituição.
- **🤝 Facilitar**: a interação direta entre estudantes e o setor empresarial local.
- **📂 Centralizar**: portfólios acadêmicos em uma vitrine digital organizada.
- **💡 Incentivar**: o empreendedorismo e a inovação dentro do ambiente escolar.

---

## 🛠️ Stack Tecnológico

Desenvolvemos com tecnologias modernas e eficientes:

| Tecnologia | Uso | 
|-----------|-----|
| **Python + Flask** | Framework web robusto para backend escalável |
| **Nix** (36.5%) | Gerenciamento de ambiente declarativo |
| **HTML** (30.4%) | Interface e documentação web |
| **Shell** (17%) | Automação e scripts de sistema |

### 🔧 Foco Principal: Python + Flask

Este projeto utiliza **Python com Flask** como framework principal, permitindo desenvolvimento ágil, eficiente e sustentável de aplicações web modernas.

---

## 🎯 Proposta:

Criar uma ponte eficiente entre o talento jovem e o investimento profissional.

---

## 🎓 Educação & Aprendizagem

Este é um projeto de **Trabalho de Conclusão de Curso (TCC)** desenvolvido em parceria com:

### 📚 **ETEC Sylvio de Mattos Carvalho**
**Localização**: Matão - SP

Estamos comprometidos com a **formação de profissionais** preparados para enfrentar os desafios tecnológicos do futuro, combinando excelência acadêmica com impacto real no mercado.
